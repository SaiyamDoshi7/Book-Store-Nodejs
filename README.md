## 📖 About The Project

Book Store CRUD Application is a web-based project developed using Node.js, Express.js, MongoDB, and EJS.

This application allows users to manage book records through CRUD operations (Create, Read, Update, Delete). Users can add new books, upload book images, view book details, edit existing records, and delete books from the database.

The application follows a structured MVC-like architecture and provides a user-friendly interface for managing books efficiently.

## 🔄 CRUD Operations

### ➕ Create

Add new books with image upload functionality.

### 📖 Read

Display all books and view individual book details.

### ✏️ Update

Edit and update existing book information.

### ❌ Delete

Remove unwanted books from the database.
